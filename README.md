# zoa

A fast, privacy-focused web utility toolkit for developers.

## Features

- **Encoder / Decoder** - Convert text between Base64, URL, and Hex encoding formats
- **Hash Generator** - Generate MD5, SHA-1, SHA-256, SHA-384, and SHA-512 hashes

## Privacy

All processing happens server-side with no data storage. Your input is never logged or persisted.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
